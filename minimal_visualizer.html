How It Works

1 Source Code

You write code in a simple language with variables, loops, and conditionals.

2 Compilation

The compiler translates your code into low-level instructions: LOAD, ADD, SUB, JUMP, etc.

3 ALU Operations

Each arithmetic operation becomes a series of logic gates: XOR, AND, OR.

For example, adding two 8-bit numbers requires 40 gates (5 per bit × 8 bits).

4 Rule 110 Execution

Each gate is a real cellular automaton. We place inputs on a tape and evolve it using Rule 110:

Rule 110: A cell's next state depends on itself and its two neighbors. The pattern 110 in binary = 6 specific rules.

5 Result

After evolution, we read the output from a specific cell position. The CA has computed the logic gate!
